BOOKSTORE16 - DETAILED PROJECT DOCUMENTATION

1. PROJECT OVERVIEW
BookStore16 is a full-stack scalable e-commerce bookstore application designed to handle real-world online bookstore workflows. It includes authentication, product management, cart system, order processing, and admin control features.

Technologies Used:
- Backend: Spring Boot (Java)
- Frontend: React + Vite
- Database: MySQL
- Authentication: JWT-based authentication
- Communication: REST APIs + WebSocket

--------------------------------------------------

2. ARCHITECTURE FLOW

Frontend (React)
   ↓
Service Layer (API calls)
   ↓
Spring Boot REST APIs
   ↓
Service Layer (Business Logic)
   ↓
Repository Layer (JPA/Hibernate)
   ↓
MySQL Database

--------------------------------------------------

3. BACKEND STRUCTURE

Key Layers:
- Controller Layer: Handles HTTP requests
- Service Layer: Contains business logic
- Repository Layer: Database interaction using JPA
- Entity Layer: Database models
- DTO Layer: Data transfer objects

Additional Features:
- Global Exception Handling using @ControllerAdvice
- Input validation using Hibernate Validator
- JWT Authentication & Role-based Authorization
- File upload handling
- Email notifications
- WebSocket for real-time order updates

--------------------------------------------------

4. FRONTEND STRUCTURE

Main folders:
- pages/
- components/
- services/
- context/
- routes/

Key Features:
- Role-based routing (Admin/User)
- Global state management using Context API
- API integration using Axios
- Reusable components (Navbar, BookCard, Spinner, etc.)
- Lazy loading and optimized rendering
- Skeleton loaders for better UX

--------------------------------------------------
